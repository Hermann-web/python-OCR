cours2: JS
Il s'agit d'un cours sur le javascript. Tu ne ne pas le suivre et apprendre le language à l'aide du cours suivant. car on parlera de variables, boucles et fonctions. Vers la fin, il y a un quizz où on fait un projet à l'aide de Node JS ( un framework utilisé pour exécuter du code javascript) mais je remarque qu'ils ont adopté un nouveau format plus simple. Tu peux laisser ce cours et lire ce site
https://openclassrooms.com/fr/courses/6175841/next-page-to-do

cours7: Bootstrap 
Bootstrap est un framework qui facilite l'écriture du CSS. Mais, passe à ce cours si tu travailles sur un projet où tu en as besoin. Tu apprendras plus vite. Même conseil que pour le framework suivant: Jquerry. Pour ma part, j'en avais besoin pour mon stage donc je suis allé sur le site w3schools.
https://www.w3schools.com/bootstrap5/

cours8: Jquerry
JQuerry est un framework qui facilite l'"criture du code javascript pour le web. Certains développeurs ne maitrisent mieux Jquerry que javascript mais dans ce cas, ce n'est pas toujours évident de traviller dans une team qui n'utilise pas Jquerry.
https://www.w3schools.com/jquery/

cours9: Django
C'est un framework Python pour faire du développement web.
J'ai suivi une partie de ce cours pour connaitre l'architecture Django (comment on crée un projet, comment les fichiers sont organisés, A quoi sert chacun d'eux). Mais, j'ai appris le reste en lisant la documentation et en visitant stackoverflow.
